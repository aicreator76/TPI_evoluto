# Catalogo DPI — Overview

### Endpoint principali
- `GET /api/dpi/csv/template`
- `POST /api/dpi/csv/save`
- `GET /api/dpi/csv/catalogo`
- `GET /api/dpi/csv/export?gruppo=...&columns=...`
- `GET /api/dpi/csv/metrics`
- `GET /api/dpi/csv/report.html`

### Note rapide
- Alias colonne: `short`, `listino`, `full`.
- `report.html` è una mini dashboard HTML pronta da linkare.
